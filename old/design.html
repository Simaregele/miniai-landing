Кейсы наших проектов

Система анализа инвойсов для логистической компании 
Проблема
Заказчик работает с большим числом иностранных поставщиков. Требуется быстро заполнять документацию на основе разнородных инвойсов (PDF, сканы, изображения). До проекта 40 сотрудников вручную извлекали данные.

Как проходила работа
1. Создание бенчмарка качества на основе обработанных и оригинальных документов.
2. Быстрый прототип подтвердил ценность решения.
3. Итеративная доработка с непрерывными экспериментами для повышения метрик.

Архитектура
• Классификация документов по языкам.
• OCR + извлечение текстовых и графических данных.
• Преобразование в Markdown и поиск похожих документов в векторной БД.
• Извлечение данных агентами LLM (Chain-of-Thought).
• Интернет-валидация ключевых полей и изображений.
• Модели: классификация по фото, регрессия артикулов, конкурентный роутер ответов.
• Пост-обработка: сбор метрик, выявление слабых полей, обратная связь для улучшения.

Итог
Время обработки документов сократилось на 40 %, компания расширила клиентскую базу и объём обрабатываемых инвойсов.
Система проверки товаров на включение в санкционные списки 
Проблема
При ввозе товаров нужно проверять, подпадают ли они под международные санкции. Списки постоянно меняются, представлены в PDF/таблицах, содержат исключения.

Исходные данные
• Описания товаров клиентов (RU/EN).
• Официальные санкционные списки.
• Таможенные справочники и внутренние базы.

Архитектура
1. Веб-скрапинг нормативных актов → OCR/парсинг таблиц.
2. Нормализация и агрегация данных.
3. Классификация товара по группам ТН ВЭД.
4. LLM анализ описания, рисков и исключений.
5. Формирование обходных маршрутов (альтернативные коды).
6. Финальная валидация специалистом.

Итог
Сокращено время обработки запроса, снижена вероятность ошибок, повышена прозрачность решений. Компания быстрее реагирует на изменения санкций.
Робот-менеджер по работе с клиентами в LinkedIn 
Задачи
Автоматизировать расширение сети контактов и коммуникации с лидами в LinkedIn.

Как работает
• Алгоритм добавляет пользователей, начиная с сети целевого лида.
• Генерирует и комментирует посты, участвует в обсуждениях.
• ML-модели адаптируют сообщения под интересы аудитории.

Эффект для клиента
• Ускоренный рост сети и узнаваемости бренда.
• Повышение качества лидогенерации.
• Экономия времени сотрудников.

Ограничения
Оценка эффективности затруднена ограничениями LinkedIn (лимиты инвайтов, длительные эксперименты).
LLM-ассистент для анализа товарных предложений конкурента 
Задачи
Собирать информацию о продуктах конкурентов и обновлениях, формировать актуальную таблицу для отдела продаж.

Подход
• Веб-краулер сайтов конкурентов.
• Извлечение данных из PDF/таблиц.
• LLM для структурирования и аналитики.

Польза
• Сокращение времени мониторинга.
• Более точные стратегические решения.
• Оптимизация маркетинговых кампаний и ценообразования.
Оптимизация поиска в магазине с 40 000 продуктов 
Задачи
Повысить релевантность поиска в крупном интернет-магазине.

Решение
• Скрипт формирует 5-7 ключевых фраз на каждый товар.
• Использует Mistral 7B (быстрая генерация) и ChatGPT-4 (улучшение качества).
• Ключевые слова добавляются в индекс поиска.

Результат
• Рост точности поиска и конверсии.
• Уменьшение нагрузки на маркетинг.
• Улучшенный пользовательский опыт.
